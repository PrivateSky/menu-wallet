If you see this page means that your wallet is not properly configured!

How to

To configure the wallet you need to create a menu.json file into path apihub-root/${wallet-name}/wallet-patch/menu.json
to overwrite the default menu.json in order to reflect your desired menu structure.

Possible causes

  • Missing menu.json file from the wallet-patch. Please check that you have a file: apihub-root/${wallet-name}/wallet-patch/menu.json
  • Trust loader was not able to retrieve info and files from the wallet-patch. Try to create a new wallet using new user details in create account page.
  • Wallet DSU suffered interventions with DSU Explorer solution or other ways and the menu.json was deleted from there.
